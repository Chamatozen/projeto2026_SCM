# Documentação do Projeto - AgroTech 2026

## Visão Geral
Sistema de Gestão Agrícola desenvolvido para auxiliar produtores rurais no monitoramento e controle de suas propriedades e safras.

## Módulos
- **main.py**: Módulo principal com menu do sistema
- **cadastro.py**: Cadastro e gestão de propriedades rurais
- **monitoramento.py**: Monitoramento de safras e geração de relatórios

## Equipe de Desenvolvimento
- Analista de Sistemas (Responsável pelo projeto)
- Desenvolvedor Backend
- Desenvolvedor Frontend
- Testador QA

## Tecnologias
- Python 3.x
- GitHub para controle de versão
- Git para gerenciamento de configuração local
